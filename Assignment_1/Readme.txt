Contents of this directory:

1. CC_0129_0837_1525_apis.py - code for all the APIs
2. locustfile.py - code for the load testing using locust
3. AreaNameEnum.csv - as provided by the instructors
4. Over and Above Specs directory - code for the load testing
5. Views directory - code for the APIs


How to Use:

1. Create a t2.micro EC2 instances on AWS
2. install all the requirements such as flask, pandas and datetime libraries
3. run app.py using the command 'sudo python3 app.py' on the EC2 instance
4. the locustfile.py can be used to perform load testing from another system such as your home desktop.
